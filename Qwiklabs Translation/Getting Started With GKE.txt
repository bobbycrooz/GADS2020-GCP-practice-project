Getting started with GKE 
---------------------------

in the cloud shell create enviroment variabke for zones
==========>export MY_ZONE = us-central1

Starting the Kubernetes Engine with cluster name webfronted1 and configre with 2 nodes
==========>gcloud container cluster create webfrontend -11zone $MY_ZONE --num-node


to check version:
==========>kubectl version

to run and deploy an an NGINX container
==========>kubectl create deploy nginx --image=nginx:1.177.10

to view the pods running the nginx container
==========>kubectl get pod 

Exposing the nginx container to the internet
==========>kubectl expose deployment nginx --port 80-typeLoadBalancer


Get the new services
==========>kubectl get services


Scale up the number of pods
==========>kubectl scale deployment niginx --replicas=3
